# AWSConnector

Helps to work with PyRDF module.



## Requirements

* JupyterLab >= 3.0

## Install

To install the extension, execute:

```bash
git clone https://github.com/CloudPyRDF/AWSConnector.git
cd AWSConnector
./install.sh
```

## Uninstall

To remove the extension, execute:

```bash
./uninstall.sh
```
